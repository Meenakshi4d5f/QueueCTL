# QueueCTL
# queuectl

A minimal, production-style **CLI-based background job queue system** with:

- Multiple workers
- Persistent storage (SQLite)
- Exponential backoff retries
- Dead Letter Queue (DLQ)
- Configurable behavior via CLI

Tech stack: **Python 3.10+**, standard library only.
